Объекты:
   >User user1 = new User("Имя № " + String.valueOf(0), "Фамилия № " +  String.valueOf(0), "Отчество № " +  String.valueOf(0), 0);
> 

>   User user2 = new User("Имя № " + String.valueOf(9999), "Фамилия № " +  String.valueOf(9999), "Отчество № " +  String.valueOf(9999), 9999);
